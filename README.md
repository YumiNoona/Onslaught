## 📘 Overview

**Shooter** is a thrilling top-down action rogue-lite arcade game where the player fights progressively difficult waves of enemies using multiple weapons while managing health, movement, and positioning. 

The project features a highly modular architecture, a robust data-driven weapon configuration system, and follows modern Godot 4 best practices. Whether you're looking for an arcade shooter to play or a polished open-source codebase to learn from, this project has it all.

---

## ✨ Key Features

### 🔫 Arsenal & Gunplay
- **Data-Driven Weapons:** 11+ distinct weapons ranging from Pistols and Shotguns to Assault Rifles, SMGs, and Sniper Rifles.
- **Dynamic Spread & Recoil:** Procedural crosshair recoil and weapon spread logic based on weapon type.
- **Visuals & Polish:** Satisfying screen shake, hit flashes, custom muzzle flashes, and bullet trails.
- **Weapon Shop:** Mid-wave shop system to purchase and upgrade your loadout using collected coins.

### 👾 Enemies & AI
- **Diverse Roster:** Mobs, Healers, Fliers, Mutants, Zombies, and Bosses.
- **Smart Spawning:** Procedural wave spawner with tier-based progression and difficulty scaling.
- **Loot Drops:** Enemies drop coins and power-ups upon defeat.

### ❤️ Progression & Systems
- **Character Selection:** Choose between multiple unique characters, each with bespoke abilities like Adrenaline Rush, Ground Slam, and Quick Dash.
- **Leveling & Perks:** Earn XP to level up mid-run and select from randomized perks to boost damage, fire rate, and more.
- **Achievements system:** In-game achievement tracking with UI popups and a dedicated viewer panel.
- **Combo System:** Maintain your kill streak to rack up massive scores.

## 🛠️ Installation & Setup

1. **Download Godot:** Ensure you have [Godot Engine 4.x](https://godotengine.org/download/) installed.
2. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-username/Shooter.git
   ```
3. **Import Project:** Open Godot, click **Import**, and select the `project.godot` file located in the root of the cloned repository.
4. **Play:** Press `F5` to run the game!
